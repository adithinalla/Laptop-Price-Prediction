# Laptop Price Prediciton 
Machine Learning project for predicting laptop prices using Python.
# Project Overview
This project uses machine learning techniques to predict laptop prices based on specifications 
# Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Scikit-learn
- Jupiter Notebook
# Features
- Data cleaning and preprocessing
- Exploratory Data Analysis(EDA)
- Feature engineering
- Machine learning model training
- Price prediction visualization
# Files Included
- Laptop_Price_Prediction.ipynb
- cleaned_laptop_data.csv
- laptop_uncleaned.csv
# Output
Achieved approximately 0.60 R² score on the test dataset
